## üí° Fitur Utama

* **Multi-Bahasa (i18n):** Mendukung terjemahan penuh dalam bahasa **Indonesia** (`id`) dan **Inggris** (`en`), memungkinkan pengguna memilih bahasa mereka dengan lancar.
* **Pengalaman Mulus:** Menggunakan library **Lenis** (`@studio-freight/lenis`) untuk efek *smooth scrolling* yang halus dan performa tinggi di seluruh halaman.
* **Animasi Dinamis:** Menerapkan animasi profesional menggunakan **GSAP** dan **Framer Motion** untuk menciptakan transisi dan interaksi yang menarik pada elemen hero dan bagian lainnya.
* **Desain Responsif:** Dibangun dengan **Tailwind CSS** untuk memastikan tata letak dan fungsionalitas optimal pada semua ukuran layar (Desktop, Tablet, Mobile).
* **Struktur Jelas:** Memuat bagian-bagian penting seperti Profil **Hero**, **Tentang Saya**, Matriks **Keahlian**, **Proyek**, **Sertifikat**, dan **CTA** (Call to Action).

## ‚öôÔ∏è Instalasi Lokal

Ikuti langkah-langkah ini untuk menjalankan proyek portofolio ini di mesin lokal Anda.

### Prasyarat

Pastikan Anda telah menginstal versi terbaru dari:
* **Node.js** (Minimal versi yang didukung oleh Next.js 15: `^18.18.0 || ^19.8.0 || >= 20.0.0`).
* **npm** atau **Yarn**

### Langkah-langkah

1.  **Clone Repositori:**
    ```bash
    git clone [https://github.com/daffaaem/daffa-portofolio-website.git](https://github.com/daffaaem/daffa-portofolio-website.git)
    cd daffa-portofolio-website
    ```

2.  **Instal Dependensi:**
    ```bash
    npm install 
    # atau
    # yarn install
    ```

3.  **Jalankan Server Pengembangan (Development Server):**
    ```bash
    npm run dev
    ```
    Proyek akan berjalan di `http://localhost:3000` (atau port lain jika 3000 sudah terpakai).

4.  **Build dan Jalankan (Production):**
    ```bash
    npm run build
    npm run start
    ```
